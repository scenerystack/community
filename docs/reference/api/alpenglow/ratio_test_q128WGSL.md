# ratio_test_q128WGSL

## Overview

Returns a range query on the rational (q128) value:

2i: totally internal (0&lt;q&lt;1)
1i: on an endpoint (q=0 or q=1)
0i: totally external (q&lt;0 or q&gt;1)
